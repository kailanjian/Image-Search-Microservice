Image Search Abstraction Layer

Based on the FreeCodeCamp challenge: https://www.freecodecamp.org/challenges/image-search-abstraction-layer

To use: submit a get request to:

  • /api/imagesearch/:search?offset=[OFFSET] (to search images, ':search' is query and [OFFSET] is the integer page of results to view)
  • /api/latest/imagesearch (to view 10 latest images)
